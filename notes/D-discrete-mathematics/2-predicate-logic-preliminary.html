谓词逻辑初步

∀n(isEven⁡(n)∧(n>2)→∃m∃k(isPrime⁡(m)∧isPrime⁡(k)∧(n=m+k)))\forall n\biggl(\operatorname{isEven}(n) \land\bigl(n>2\bigr) \rightarrow \exists m \exists k\Bigl(\operatorname{isPrime}(m) \land \operatorname{isPrime}(k) \land\bigl(n=m+k\bigr)\Bigr)\biggr) ∀n(isEven(n)∧(n>2)→∃m∃k(isPrime(m)∧isPrime(k)∧(n=m+k)))

哥德巴赫猜想无法用命题逻辑表达，因为 nnn 等是变量。

谓词（Predicate）

  • 一元谓词 P(x)P(x)P(x)：给定 xxx，P(x)P(x)P(x) 为真或假。
  • 二元谓词 Q(x,y)Q(x,y)Q(x,y)：给定 xxx 和 yyy，Q(x,y)Q(x,y)Q(x,y) 为真或假。
  • ……

原子陈述

  • P(t1,⋯ ,tn)P(t_1, \cdots, t_n)P(t1​,⋯,tn​)
  • 其中 PPP 是 nnn 元谓词，t1,⋯ ,tnt_1, \cdots, t_nt1​,⋯,tn​ 是常量、变量或函数取值

逻辑公式（陈述，Formula）

  • 原子陈述是逻辑公式
  • 若 PPP 是逻辑公式，xxx 是自由变量，则 ∀x P\forall x\, P∀xP 和 ∃x P\exists x\, P∃xP 是逻辑公式
  • 若 PPP 和 QQQ 是逻辑公式，则 ¬P, P∧Q, P∨Q, P→Q\lnot P,\, P \land Q,\, P \lor Q,\, P \to Q¬P,P∧Q,P∨Q,P→Q 是逻辑公式

量词的优先级高于其它逻辑运算符。

  • 约束变元
    • ∀x∃y(y>x)\forall x \exists y (y > x)∀x∃y(y>x)（即 ∀x(∃y(y>x))\forall x (\exists y (y > x))∀x(∃y(y>x))）中，x,yx, yx,y 是约束变元
  • 自由变元
    • ∃(y>x)∧(x+2>y)\exists (y > x) \land (x + 2 > y)∃(y>x)∧(x+2>y) 中，xxx 与后面的 yyy 是自由变元（后面是自由变元，因为前面的 ∃\exists ∃ 作用不到后面的 yyy）
  • 量词作用域
    • ∀x∃y(y>x)\forall x \exists y (y > x)∀x∃y(y>x) 中，∀x\forall x∀x 的作用域是 (∃y(y>x))(\exists y (y > x))(∃y(y>x))，∃y\exists y∃y 的作用域是 (y>x)(y > x)(y>x)
  • 重命名（对约束变元）
    ∃(y>x)∧(x+2>y)≡∃z(z>x)∧(x+2>y)\exists (y > x) \land (x + 2 > y) \equiv \exists z (z > x) \land (x + 2 > y)∃(y>x)∧(x+2>y)≡∃z(z>x)∧(x+2>y)

量词：

  • 全称量词 ∀x P(x)\forall x \, P(x)∀xP(x)：对所有 xxx，P(x)P(x)P(x) 为真
  • 存在量词 ∃x P(x)\exists x \, P(x)∃xP(x)：存在 xxx，使得 P(x)P(x)P(x) 为真

多个量词并用：

  • ∀x∀y P(x,y)≡∀y∀x P(x,y)\forall x \forall y \, P(x, y) \equiv \forall y \forall x \, P(x, y)∀x∀yP(x,y)≡∀y∀xP(x,y)
  • ∃x∃y P(x,y)≡∃y∃x P(x,y)\exists x \exists y \, P(x, y) \equiv \exists y \exists x \, P(x, y)∃x∃yP(x,y)≡∃y∃xP(x,y)
  • ∀x∃y P(x,y)≢∃y∀x P(x,y)\forall x \exists y \, P(x, y) \not\equiv \exists y \forall x \, P(x, y)∀x∃yP(x,y)≡∃y∀xP(x,y)

常用逻辑等价式/蕴含式（RRR 中不含自由变量 xxx）

  • ¬∀x P(x)≡∃x ¬P(x)\lnot \forall x \, P(x) \equiv \exists x \, \lnot P(x)¬∀xP(x)≡∃x¬P(x)
  • ¬∃x P(x)≡∀x ¬P(x)\lnot \exists x \, P(x) \equiv \forall x \, \lnot P(x)¬∃xP(x)≡∀x¬P(x)
  • ∀x(P(x)∧Q(x))≡∀x P(x)∧∀x Q(x)\forall x \bigl(P(x) \land Q(x)\bigr) \equiv \forall x \, P(x) \land \forall x \, Q(x)∀x(P(x)∧Q(x))≡∀xP(x)∧∀xQ(x)
  • ∃x(P(x)∨Q(x))≡∃x P(x)∨∃x Q(x)\exists x \bigl(P(x) \lor Q(x)\bigr) \equiv \exists x \, P(x) \lor \exists x \, Q(x)∃x(P(x)∨Q(x))≡∃xP(x)∨∃xQ(x)
  • ∀x P(x)∨∀x Q(x)⊨∀x(P(x)∨Q(x))\forall x \, P(x) \lor \forall x \, Q(x) \models \forall x \bigl(P(x) \lor Q(x)\bigr)∀xP(x)∨∀xQ(x)⊨∀x(P(x)∨Q(x))（奇偶）
  • ∃(P(x)∧Q(x))⊨∃x P(x)∧∃x Q(x)\exists \bigl(P(x) \land Q(x)\bigr) \models \exists x \, P(x) \land \exists x \, Q(x)∃(P(x)∧Q(x))⊨∃xP(x)∧∃xQ(x)
  • ∀x(P(x)∨R)≡(∀x P(x))∨R\forall x \bigl(P(x) \lor R\bigr) \equiv \bigl(\forall x \, P(x)\bigr) \lor R∀x(P(x)∨R)≡(∀xP(x))∨R
  • ∃x(P(x)∧R)≡(∃x P(x))∧R\exists x \bigl(P(x) \land R\bigr) \equiv \bigl(\exists x \, P(x)\bigr) \land R∃x(P(x)∧R)≡(∃xP(x))∧R
  • ∀x(R→P(x))≡R→∀x P(x)\forall x \bigl(R \to P(x)\bigr) \equiv R \to \forall x \, P(x)∀x(R→P(x))≡R→∀xP(x)（即 ∀x(¬R∨P(x))≡¬R∨(∀x P(x))\forall x \bigl(\lnot R \lor P(x)\bigr) \equiv \lnot R \lor \bigl(\forall x \, P(x)\bigr)∀x(¬R∨P(x))≡¬R∨(∀xP(x))）
  • ∃x(R∧P(x))≡R∧∃x P(x)\exists x \bigl(R \land P(x)\bigr) \equiv R \land \exists x \, P(x)∃x(R∧P(x))≡R∧∃xP(x)
  • ∀x(P(x)→R)≡(∃x P(x))→R\forall x \bigl(P(x) \to R\bigr) \equiv \bigl(\exists x \, P(x)\bigr) \to R∀x(P(x)→R)≡(∃xP(x))→R
  • ∃x(P(x)→R)≡(∀x P(x))→R\exists x \bigl(P(x) \to R\bigr) \equiv \bigl(\forall x \, P(x)\bigr) \to R∃x(P(x)→R)≡(∀xP(x))→R（即 ∃x(P(x)→R)≡(∃x¬P(x))∨R≡¬(∀x P(x))∨R\exists x \bigl(P(x) \to R\bigr) \equiv \bigl(\exists x \lnot P(x)\bigr) \lor R \equiv \lnot \bigl(\forall x \, P(x)\bigr) \lor R∃x(P(x)→R)≡(∃x¬P(x))∨R≡¬(∀xP(x))∨R）

前束范式（Prenex Normal Form）

∀x(x≤0∨∃y(y>0∧x=y2))≡∀x∃y(x≤0∨(y>0∧x=y2))\forall x\left(x \leq 0 \lor \exists y\left(y>0 \land x=y^{2}\right)\right) \equiv \forall x \exists y\left(x \leq 0 \lor\left(y>0 \land x=y^{2}\right)\right) ∀x(x≤0∨∃y(y>0∧x=y2))≡∀x∃y(x≤0∨(y>0∧x=y2))
