集合定义

直观定义：一个集合是一组无序的对象，这些对象称为这个集合的元素或成员。a∈Aa \in Aa∈A 表示 aaa 是集合 AAA 的一个成员，a∉Aa \notin Aa∈/A 表示 aaa 不是 AAA 的成员。

康托尔（Georg Cantor）：A set is a collection into a whole of definite, distinct objects of our intuition or our thought. The objects are called elements (member) of the set.

朴素集合论高中学过，就懒得记了……

集合关系

集合相等当且仅当它们有相同的元素。

A=BA = BA=B 当且仅当 ∀x(x∈A↔x∈B)\forall x (x \in A \leftrightarrow x \in B)∀x(x∈A↔x∈B)。

集合 AAA 称为集合 BBB 的子集，当且仅当 ∀x(x∈A→x∈B)\forall x (x \in A \rightarrow x \in B)∀x(x∈A→x∈B)，记作 A⊆BA \subseteq BA⊆B。

如果 A⊆BA \subseteq BA⊆B 且 A≠BA \neq BA=B，则称 AAA 是 BBB 的真子集，记作 A⊂BA \subset BA⊂B。

从而可知 A=B  ⟺  A⊆B∧B⊆AA = B \iff A \subseteq B \land B \subseteq AA=B⟺A⊆B∧B⊆A。

X⊆Y∧Y⊆Z  ⟹  X⊆ZX \subseteq Y \land Y \subseteq Z \implies X \subseteq Z X⊆Y∧Y⊆Z⟹X⊆Z

集合大小

对于有限集合，若集合 SSS 恰有 n∈Nn \in \Nn∈N 个不同的元素，则称 SSS 为有限集合（finite set），并记 ∣S∣=n|S| = n∣S∣=n 为 SSS 的基数。

如果一个集合不是有限集合，则称其为无限集合（infinite set）。

特别地，空集 ∅\empty∅ 是唯一的零集。

幂集

集合 SSS 的幂集（power set）是 SSS 的所有子集的集合，记作 P(S)\mathcal{P}(S)P(S)。即 P(S)={A∣A⊆S}\mathcal{P}(S) = \{A \mid A \subseteq S\}P(S)={A∣A⊆S}。

显然，∣P(S)∣=2∣S∣|\mathcal{P}(S)| = 2^{|S|}∣P(S)∣=2∣S∣，因此 SSS 的幂集还可以记作 2S2^S2S。

A⊂B  ⟺  P(A)⊆P(B)A \subset B \iff \mathcal{P}(A) \subseteq \mathcal{P}(B) A⊂B⟺P(A)⊆P(B)

集合运算

  • 并集（union）：A∪B={x∣x∈A∨x∈B}A \cup B = \{x \mid x \in A \lor x \in B\}A∪B={x∣x∈A∨x∈B}。
  • 交集（intersection）：A∩B={x∣x∈A∧x∈B}A \cap B = \{x \mid x \in A \land x \in B\}A∩B={x∣x∈A∧x∈B}。
  • 差集（difference）：A−B={x∣x∈A∧x∉B}A - B = \{x \mid x \in A \land x \notin B\}A−B={x∣x∈A∧x∈/B}，也称作 BBB 对于 AAA的补集。
    • 对于全集 UUU，U−AU - AU−A 称为 AAA 的补集，记作 ∼B\sim B∼B
  • 对称差（symmetric difference）：A⊕B=(A−B)∪(B−A)A \oplus B = (A - B) \cup (B - A)A⊕B=(A−B)∪(B−A)[1]。
    • A⊕B=(A∪B)−(A∩B)A \oplus B = (A \cup B) - (A \cap B)A⊕B=(A∪B)−(A∩B)
  • 广义并（generalized union）：⋃A={x∣∃y(y∈A∧x∈y)}\bigcup A = \left\lbrace x \mid \exists y (y \in A \land x \in y) \right\rbrace⋃A={x∣∃y(y∈A∧x∈y)}。
    • 例如 {{1,2},{2,3}}\left\lbrace \{1, 2\}, \{2, 3\} \right\rbrace{{1,2},{2,3}} 的广义并是 {1,2,3}\{1, 2, 3\}{1,2,3}。
  • 广义交（generalized intersection）：⋂A={x∣∀y(y∈A→x∈y)}\bigcap A = \left\lbrace x \mid \forall y (y \in A \to x \in y) \right\rbrace⋂A={x∣∀y(y∈A→x∈y)}（其中 A≠∅A \ne \emptyA=∅）
    • 例如 {{1,2},{2,3}}\left\lbrace \{1, 2\}, \{2, 3\} \right\rbrace{{1,2},{2,3}} 的广义交是 {2}\{2\}{2}。
    • ⋂∅\bigcap \empty⋂∅ 无意义。

A∪BA \cup BA∪B 是 AAA 和 BBB 的最小上界，A∩BA \cap BA∩B 是 AAA 和 BBB 的最大下界。

恒等式：

  • 恒等律
    • A∪∅=AA \cup \empty = AA∪∅=A
    • A∩U=AA \cap U = AA∩U=A
  • 支配律
    • A∪U=AA \cup U = AA∪U=A
    • A∩∅=∅A \cap \empty = \emptyA∩∅=∅
  • 幂等律
    • A∪A=AA \cup A = AA∪A=A
    • A∩A=AA \cap A = AA∩A=A
  • 补集律：∼(∼A)=A\sim (\sim A) = A∼(∼A)=A
  • 交换律
    • A∪B=B∪AA \cup B = B \cup AA∪B=B∪A
    • A∩B=B∩AA \cap B = B \cap AA∩B=B∩A
  • 结合律
    • (A∪B)∪C=A∪(B∪C)(A \cup B) \cup C = A \cup (B \cup C)(A∪B)∪C=A∪(B∪C)
    • (A∩B)∩C=A∩(B∩C)(A \cap B) \cap C = A \cap (B \cap C)(A∩B)∩C=A∩(B∩C)
  • 分配律
    • A∪(B∩C)=(A∪B)∩(A∪C)A \cup (B \cap C) = (A \cup B) \cap (A \cup C)A∪(B∩C)=(A∪B)∩(A∪C)
    • A∩(B∪C)=(A∩B)∪(A∩C)A \cap (B \cup C) = (A \cap B) \cup (A \cap C)A∩(B∪C)=(A∩B)∪(A∩C)
  • 德摩根律
    • ∼(A∪B)=∼A∩∼B\sim (A \cup B) = \sim A \cap \sim B∼(A∪B)=∼A∩∼B
    • ∼(A∩B)=∼A∪∼B\sim (A \cap B) = \sim A \cup \sim B∼(A∩B)=∼A∪∼B
  • 吸收律
    • A∪(A∩B)=AA \cup (A \cap B) = AA∪(A∩B)=A
    • A∩(A∪B)=AA \cap (A \cup B) = AA∩(A∪B)=A
  • 补律
    • A∪∼A=UA \cup \sim A = UA∪∼A=U
    • A∩∼A=∅A \cap \sim A = \emptyA∩∼A=∅

笛卡尔乘积

有序偶（ordered pair）：(a,b)(a, b)(a,b) 表示 aaa 和 bbb 的有序偶。(a,b)=(x,y)  ⟺  a=x∧b=y(a, b) = (x, y) \iff a = x \land b = y(a,b)=(x,y)⟺a=x∧b=y。

  • (a,b)≜{{a},{a,b}}(a, b) \triangleq \left\lbrace \{a\}, \{a, b\} \right\rbrace(a,b)≜{{a},{a,b}}

笛卡尔乘积（Cartesian product）：A×B={(a,b)∣a∈A∧b∈B}A \times B = \{(a, b) \mid a \in A \land b \in B\}A×B={(a,b)∣a∈A∧b∈B}。

  • A×BA \times BA×B 不一定等于 B×AB \times AB×A，A×B=B×AA \times B = B \times AA×B=B×A 当且仅当 A=BA = BA=B 或 A=∅A = \emptyA=∅ 或 B=∅B = \emptyB=∅。
