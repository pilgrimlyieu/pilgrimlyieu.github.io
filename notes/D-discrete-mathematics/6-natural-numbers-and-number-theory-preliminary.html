自然数

皮亚诺公理（Peano axioms for natural numbers）

  1. 000 是自然数。
  2. 每个自然数 xxx 都有一个自然数后继 S(x)S(x)S(x)。
  3. 000 不是任何自然数的后继。
  4. 不同的自然数有不同的后继（即 S(x)=S(y)→x=yS(x) = S(y) \to x = yS(x)=S(y)→x=y）。
  5. 如果一个集合 AAA 包含 000，并且对于每个 x∈Sx \in Sx∈S，S(x)∈AS(x) \in AS(x)∈A，那么 AAA 包含所有自然数。

形式化写法：

  1. 0∈N0 \in \N0∈N
  2. x∈N→S(x)∈Nx \in \N \to S(x) \in \Nx∈N→S(x)∈N
  3. x∈N→S(x)≠0x \in \N \to S(x) \ne 0x∈N→S(x)=0
  4. x∈N∧y∈N∧S(x)=S(y)→x=yx \in \N \land y \in \N \land S(x) = S(y) \to x = yx∈N∧y∈N∧S(x)=S(y)→x=y
  5. 0∈M∧∀x(x∈→S(x)∈M)→N⊆M for any property M0 \in M \land \forall x(x \in \to S(x) \in M) \to \N \subseteq M \text{ for any property } M0∈M∧∀x(x∈→S(x)∈M)→N⊆M for any property M

由皮亚诺公理可知，自然数 N\NN 集合只取决于 000 与 SSS。从集合和集合运算出发，构造一个结构 ⟨0,N,S⟩\left\langle 0, \N, S \right\rangle⟨0,N,S⟩ 满足皮亚诺公理。

冯·诺伊曼构造（von Neumann construction）

  • 0=∅0 = \empty0=∅
  • S(x)=x∪{x}S(x) = x \cup \left\lbrace x \right\rbraceS(x)=x∪{x}

有意思的是，card⁡N=n\operatorname{card} N = ncardN=n（NNN 代表冯·诺依曼构造下 nnn 的表示，意会即可）。同时也有 N={0,1,⋯ ,n−1}N = \left\lbrace 0, 1, \cdots, n - 1 \right\rbraceN={0,1,⋯,n−1}。

设 aaa 为集合，称 a∪{a}a \cup \left\lbrace a \right\rbracea∪{a} 为 aaa 的后继。记为 S(a)S(a)S(a) 或 a+a^{+}a+。

设 AAA 是集合，若 AAA 满足下列条件，则称 AAA 是归纳集：

  • ∅∈A\empty \in A∅∈A
  • ∀a(a∈A→S(a)∈A)\forall a (a \in A \to S(a) \in A)∀a(a∈A→S(a)∈A)

从而定义自然数集合 N\NN 为所有归纳集的交集，即

N={∅,{∅},{∅,{∅}},⋯ }\N = \left\lbrace \empty, \left\lbrace \empty \right\rbrace, \left\lbrace \empty, \left\lbrace \empty \right\rbrace \right\rbrace, \cdots \right\rbrace N={∅,{∅},{∅,{∅}},⋯}

也就可以定义加法：对任意自然数 m, nm,\, nm,n

  • m+0=mm + 0 = mm+0=m
  • m+S(n)=S(m+n)m + S(n) = S(m + n)m+S(n)=S(m+n)

整除

设 a, ba,\, ba,b 为整数（a≠0a\ne 0a=0），若存在整数 ccc 使得 b=acb = acb=ac，则称 aaa 整除 bbb（或 bbb 能被 aaa 整除），记作 a∣ba \mid ba∣b。

设 a, b, ca,\, b,\, ca,b,c 是整数，a≠0a \ne 0a=0，则

  • 若 a∣ba \mid ba∣b，则 a∣bca \mid bca∣bc
  • 若 a∣ba \mid ba∣b 且 b∣cb \mid cb∣c，则 a∣ca \mid ca∣c
  • 若 a∣ba \mid ba∣b 且 a∣ca \mid ca∣c，则 a∣(b+c)a \mid (b + c)a∣(b+c)
    • 推广：若 a∣ba \mid ba∣b 且 a∣ca \mid ca∣c，则 a∣(mb+nc)a \mid (mb + nc)a∣(mb+nc)，其中 m, nm,\, nm,n 为任意整数

最大公约数

能整除两个整数（不全为 000）的最大整数称为这两个整数的最大公约数（Greatest Common Divisor），记作 gcd⁡(a,b)\gcd(a, b)gcd(a,b)。即 gcd⁡(a,b)=max⁡{d∈Z+∣d∣a∧d∣b}(a≠0∨b≠0)\gcd(a, b) = \max\left\lbrace d \in \Z^{+} \mid d \mid a \land d \mid b \right\rbrace\quad (a \ne 0 \lor b \ne 0)gcd(a,b)=max{d∈Z+∣d∣a∧d∣b}(a=0∨b=0)。

即 a, ba,\, ba,b 互素   ⟺  gcd⁡(a,b)=1\iff \gcd(a, b) = 1⟺gcd(a,b)=1。

辗转相除法：设 a, ba,\, ba,b 为整数（不全为 000），则有 gcd⁡(a,b)=gcd⁡(b,a mod b)\gcd(a, b) = \gcd(b, a \bmod b)gcd(a,b)=gcd(b,amodb)。
